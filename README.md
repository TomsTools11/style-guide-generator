# Style Guide Generator

Generate comprehensive website style guides and design systems from URLs, screenshots, and existing documentation. A Cowork skill that produces professional PDF outputs following industry-standard style guide structure.

## Overview

This skill analyzes provided URLs, screenshots, uploaded files, and user requirements to generate professionally formatted style guides. The output serves as the single source of truth for design and development teams.

## Features

- Extracts design elements from live websites (colors, typography, spacing)
- Analyzes screenshots and uploaded documentation
- Follows industry-standard style guide structure
- Generates professional PDF output
- Includes accessibility (WCAG 2.1 AA) guidelines

## Contents

- `SKILL.md` — Core skill instructions and workflow
- `assets/template.pdf` — Reference template for style guide formatting
- `references/design_system_examples.md` — Best practices from leading design systems
- `scripts/analyze_website.py` — Python utility for extracting design tokens from websites

## Usage

Place this skill directory in your Cowork skills folder and invoke when you need to create a style guide from a website URL, screenshots, or existing brand materials.
